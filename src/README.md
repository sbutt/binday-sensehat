# Binday Sense HAT

Offline Raspberry Pi Sense HAT bin-day reminder system.

## Features

- Sense HAT LED reminders
- Flashing reminder after 7pm
- Joystick acknowledgement
- Persistent acknowledgement state
- UK bank holiday handling
- Offline operation
- Rotating logs

## Hardware

- Raspberry Pi
- Sense HAT

## Run

```bash
python3 src/main.py
